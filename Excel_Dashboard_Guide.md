Excel Dashboard Guide

1. Open Excel and load data/business_sales_data.csv into a table.
2. Use Pivot Tables to create:
   - Revenue by Month (rows) and Region (columns)
   - Profit by ProductCategory
   - Top products list (filter Top 10)
3. Create charts from the pivot tables (line chart for trend, bar charts for comparisons).
4. Add slicers for Region and ProductCategory for interactivity.
